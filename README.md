# 724 Restaurant And Bar

Ordering website for **724 Restaurant And Bar** — NERDC Rd, Agidingbi, Ikeja, Lagos. Open 24 hours, 7 days a week. Grilled fish, Naija classics (pounded yam & egusi, isi ewu), and a full bar. Dine-in, takeaway and delivery.

Built with Next.js 15, React 19, Tailwind CSS and shadcn/ui. Brand color: `#580b1b`.

To get started, run `npm run dev` and take a look at `src/app/page.tsx`.
