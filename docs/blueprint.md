# **App Name**: 724 Restaurant And Bar

## Core Features:

- Cinematic Menu Navigator: A high-performance interactive menu with category filters for Grills, Soups & Specials, and Bar & Drinks, featuring smooth Framer Motion transitions and touch-friendly food cards.
- The 724 Taste Guide: An AI-powered tool that acts as a flavor sommelier, using reasoning to recommend specific meal pairings and spice levels based on the user's past selections or current cravings.
- Luxury Mobile Checkout: A streamlined mobile-app checkout experience with floating action buttons, promo code integration, and a glassmorphic delivery cost calculator.
- Real-time Order Status Simulation: A cinematic delivery progress interface that visually tracks 'Order Confirmed', 'In the Kitchen', and 'Out for Delivery' with rich animation counters.
- Lagos Lounge Visual Hub: A narrative-driven section highlighting the 24/7 bar-and-restaurant vibe with an Instagram-style food gallery designed for visual impact.
- Floating Concierge Navigation: A sticky mobile-first bottom navigation bar and a floating WhatsApp CTA button for constant brand access.

## Style Guidelines:

- Primary: Deep Wine (#580b1b) used for action buttons and badges — the 724 brand color. Background: Warm off-white for a calm, premium feel.
- Accents: Gold (#F4B400) for highlights and premium tags; Warm Cream (#FFF5E8) for soft surfaces.
- Font pairing: 'Playfair Display' (serif) for elegant, high-contrast headings to evoke a premium lounge, and 'Inter' (sans-serif) for neutral, highly legible body text and UI controls.
- Minimalist and thin-stroke Lucide React icons, using the Gold accent color for subtle navigation highlights.
- Mobile-first immersive experience with rounded cards (24px radius), generous vertical padding, and a sticky mobile footer reminiscent of premium native apps.
- Smooth parallax effects for hero backgrounds, staggered list entry for food items, and floating motion for checkout CTAs to simulate an app-like fluidity.
